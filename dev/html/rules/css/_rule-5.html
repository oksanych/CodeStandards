Variables

  • Add to variables colors (more about colors in variables), font-sizes, font-families, breakpoints etc.
  • Separate the semantic blocks with variable empty lines and comments.
    
    //----------------------------
    // Color pallet
    //----------------------------

    // Gray
    $gray33:                      #333;
    $gray46:                      #464646;

    // Red
    $red:                         #ff0000;

    // Green
    $green:                       #2ECC40;

    // Blue
    $blue:                        #0074D9;


    //----------------------------
    // Color variables
    //----------------------------
    $color-main:                  $blue;
    $color-success:               $green;
    $color-danger:                $red;

    //----------------------------
    // Font
    //----------------------------

    // Font-size
    $font-size-xs:                1.2rem;
    $font-size-sm:                1.4rem;
    $font-size-md:                1.6rem;
    $font-size-lg:                1.8rem;
    $font-size-xlg:               2rem;

    // Font-family
    $font-family:                 'Roboto', Arial, sans-serif;
    $font-family--secondary:      'Helvetica', Arial, sans-serif;

    //----------------------------
    // Breakpoints
    //----------------------------
    $screen-xs:                   320px;
    $screen-sm:                   768px;
    $screen-md:                   1170px;
    $screen-lg:                   1366px;

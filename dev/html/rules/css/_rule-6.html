Sass placeholders

  • Placeholder selectors will not show up in the generated CSS, only the selectors that extend them will be included in the output. You can create separate file with placeholders and called it helpers and using this helpers in the project. For better understanding read this article - Understanding placeholder selectors 
    
    %list-reset {
      transition: background-color ease .2s;
      margin: 0 .5em;
    }

    .product-list {
      @extend %list-reset;
      /* error specific styles... */
    }

    .user-list {
      @extend %list-reset;
      /* info specific styles... */
    }

@media

  • Insert @media to the selectors, and not vice versa.
  • Don't put @media into each other.
  • Prefer the path of mobile-first, avoid specifying the @media-condition max-width in favor of min-width.
  • Write @media near each other, don't write selectors between them.
  • Write the conditions for @media where you use @media, do not write conditions in variables. 
    
    .promo {
      display: block;

      // Right: the condition is clear, in the variable only the width
      @media (min-width: $screen-md) {
        display: none;

        &__img {
          ...
        }
      }

      @media (min-width: $screen-lg) {
        display: block;
      }

      // Wrong: the condition is entirely put into a variable (non-obviousness)
      @media ($mobile-width) {
        display: block;
      }
    }

Mixing - combining on one tag classes BEM-block and BEM-element

The combination is possible in any combination: BEM-block + BEM-element, BEM-block + BEM-block, BEM-element + BEM-element.

  • Classes of BEM blocks should be written first.
  • Add some style properties needed only at the location of the addition (using the modifier is irrational). Example: for .btn inside .page-header, an external left margin of 37 pixels is required. You can add an additional class .page-header__btn for the .btn tag and indent with this selector. This is a normal practice, you can safely use it.
    
        <!-- Example-1 -->
        <div class="page-header">
            <div class="page-header__nav-wrap">...</div>
            <a class="btn  page-header__btn" href="/buy">...</a>
        </div>

        <!-- Example-2 -->
        <div class="slider">
          <div class="promo  slider__inner">
            <div class="promo__item  slider__item">
              ...
            </div>
          </div>
        </div>
